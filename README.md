#音樂功能 music commands

\play        

             在一個語音頻道播放音樂(只能放youtube)
             
\resume

             繼續播放
             
\pause

             暫停播放
             
\stop

             完全停止播放
             *用resume不可回復
             
\join        

             加入你所在的語音頻道
             
\leave       

             離開你所在的語音頻道
             
------------------------
