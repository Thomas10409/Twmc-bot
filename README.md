#功能 commands

\play        

             在一個語音頻道播放音樂(只能放youtube)
             
\join        

             加入你所在的語音頻道
             
\leave       

             離開你所在的語音頻道
             
\help        

             指令列表
             
\covid       

             獲取昨日covid確診數(僅限台灣地區)
             
\ㄐㄐ        

             用用看你就會知道了
             
\rickroll    

             恭喜你被rickroll了嘿嘿
             
\ping        

             pong!獲得機器人延遲
